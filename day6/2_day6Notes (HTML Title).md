# HTML Page Title

Every webpage should have a **page title** that describes the **content and purpose of the page**.

The title appears in:

* Browser tabs
* Browser title bar
* Bookmarks/Favorites
* Search engine results

---

# The `<title>` Element

The `<title>` element defines the **title of the HTML document**.

It must be placed inside the **`<head>` section**.

Example:

```html id="6c2r06"
<!DOCTYPE html>
<html>
<head>
  <title>HTML Tutorial</title>
</head>
<body>

The content of the document...

</body>
</html>
```

Result:

* The browser tab will display **HTML Tutorial**.

---

# Where the Page Title Appears

The title is used in several places:

1. **Browser Tab**

Example:

```
HTML Tutorial | Browser Tab
```

2. **Bookmarks / Favorites**

When a user bookmarks the page, the **title becomes the bookmark name**.

3. **Search Engine Results**

Search engines like Google use the title as the **main clickable link**.

---

# Importance for SEO

The page title is important for **Search Engine Optimization (SEO)**.

Search engines use the title to:

* understand page content
* rank pages in search results
* display page links

Example search result:

```
HTML Tutorial
www.example.com
Learn HTML basics including tags, images, and links.
```

---

# Characteristics of a Good Page Title

A good title should be:

* **Clear**
* **Accurate**
* **Descriptive**
* **Relevant to the page content**

Example:

Good:

```
HTML Images Tutorial
```

Bad:

```
Page 1
```

---

# HTML Title Tag

| Tag       | Description                            |
| --------- | -------------------------------------- |
| `<title>` | Defines the title of the HTML document |

---

# Key Points

* Every webpage should have a **title**.
* The `<title>` element must be inside the **`<head>` section**.
* The title appears in **browser tabs, bookmarks, and search results**.
* A clear title improves **SEO and user experience**.
